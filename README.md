## Run Locally

Clone the project

```bash
  git clone https://github.com/akhileshjavalagi/desert-cart-clone
```

Install dependencies

```bash
  npm init
```

Start the server

```bash
  npm run start
```


## Tech Stack

**Client:** HTML, CSS, JavaScript

**Server:** Node, Express
